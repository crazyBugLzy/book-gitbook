  • 目录结构

目录结构

.
├── book.json
├── README.md
├── SUMMARY.md
├── asserts
├── styles
    ├── website.css
├── docs
    ├── chapter-1/
    |   ├── README.md
    |   └── something.md
    └── chapter-2/
        ├── README.md
        └── something.md
  • README.md和SUMMARY.md是执行gitbook init后自动生成的，book.json、styles/website.css和asserts都是个性化设置需要的。
  • asserts这个目录不是gitbook规定的，主要是本人用来放置资源文件如视频、图片等。
  • docs 目录用于放置各个章节书籍，可以根据SUMMARY.md中设置的目录结构自动生成。同样的，不同人习惯的命名和放置位置也千差万别。SUMMARY.md示例如下：
# Summary

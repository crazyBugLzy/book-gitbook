命令

  • 初始化：gitbook init
  • 插件安装：gitbook install
  • 构建静态HTML：gitbook build
  • 构建+运行：gitbook serve
  • 查看版本号：gitbook -V
  • 列出所有命令：gitbook help
  • 帮助信息：gitbook --help
  • 列出本地所有的gitbook版本：gitbook ls
  • 列出远程可用的gitbook版本：gitbook ls-remote
  • 安装对应的gitbook版本：gitbook fetch 标签/版本号
  • 更新到gitbook的版本：gitbook update
  • 卸载版本：gitbook uninstall 2.0.1
  • 指定log的级别：gitbook build/serve --log=debug
  • 输出错误信息：gitbook build --debug

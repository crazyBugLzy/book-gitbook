功能插件

1. 评论 Disqus

插件地址

"plugins": [
   "disqus"
],
"pluginsConfig": {
   "disqus": {
       "shortName": "gitbookuse"
   }
}

[warning] For warning
需要现在disqus网站上注册，然后才能进行评论。在公司内部竟然访问不了 disqus网站，所以暂时没有效果。

2. 跳转至修改页面 edit-link

"plugins": [
    "edit-link"
]
"pluginsConfig": {
    "edit-link": {
        "base": "https://github.com/zhangjikai/gitbook-use/edit/master",
        "label": "Edit This Page"
    }
}

4. 播放本地视频 Local Video

"plugins": [ "local-video" ]

使用示例：为了使视频可以自适应，我们指定视频的width为100%，并设置宽高比为16:9，如下面所示

6. 文件代码引入 include-codeblock

使用代码块的格式显示所包含文件的内容. 该文件必须存在。插件提供了一些配置，可以区插件官网查看。如果同时使用 ace 和本插件，本插件要在 ace 插件前面加载。 插件地址

{
    "plugins": [
        "include-codeblock"
    ],
    "pluginsConfig": {
        "include-codeblock": {
            "template": "ace",
            "unindent": "true",
            "theme": "monokai"
        }
    }
}

使用示例：

pre[class*="language-"] { border: none; background-color: #f7f7f7; font-size: 1em; line-height: 1.2em; }

7. 搜索插件

7.1. search Pro

支持中文搜索, 需要将默认的search插件去掉, 注意: 如果标题中有包含的关键字, 标题的样式会有所变化 插件地址

"plugins": [
    "-search",
    "search-pro"
],
"pluginsConfig": {
    "search-pro": {
        "cutWordLib": "nodejieba",
        "defineWord" : ["Gitbook Use"]
    }
}

7.2. search-plus

"plugins": [
    "-search",
    "search-plus@^0.0.11"
]

安装

详细指南可以参考 gitbook 文档

1. 需求

安装Gitbook非常简单，只有两个要求：

  • NodeJS(v4.0.0或者以上)
  • Windows, Linux, Unix, or Mac OS X

2. 安装NPM

最好的安装Gitbook的方式就是通过NPM进行，在客户端中，简单的运行以下指令即可完成安装：

$ npm install gitbook-cli -g

3. 测试安装成功

打开命令行，输入以下命令，看到Gitbook的安装版本即安装成功。

$ gitbook -V
